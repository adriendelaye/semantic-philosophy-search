A semantic search engine built on philosophical texts from the Stanford Encyclopedia of Philosophy.

## Features

- Web scraping of philosophy articles
- Linguistic preprocessing with spaCy
- Sentence embeddings with SentenceTransformers
- Fast vector search with FAISS
- Streamlit interface

## Tech Stack

- spaCy
- SentenceTransformers
- FAISS
- Streamlit

## Run the project

```bash
pip install -r requirements.txt
python -m spacy download en_core_web_sm
